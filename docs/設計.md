### SoftmaxWithLossの求め方
1. tsがone-hotベクトルの時(N, T, D)形状の時、1が立っているインデックスを取得し、二次元配列にする
```
if ts.ndim == 3:  # 教師ラベルがone-hotベクトルの場合
    ts = ts.argmax(axis=2)
```

2. tsと同じ形状のマスクを作成する。なお、マスクを作るときは、tsがマスクナンバーと等しくないとき、要素をTrueとする
```
 mask = (ts != self.ignore_label)
```

3. xsの形状を(N*T, V)に形状変換

4. ts, maskをフラットに形状変換

5. xsのsoftmaxを計算。softmax計算後もxsの形状は変化しない
```
ys = softmax(xs)
```

6. ys(N*T, V)からtsが持つインデックスの列を選択し抽出 → (N*T)
```
ls = np.log(ys[np.arange(N * T), ts])
```

7. マスクがオンになっている要素のみそのまま値を出力。それ以外は0にする
```
ls *= mask
```

8.  損失を1つにまとめる
```
loss = -np.sum(ls)
```

9. 有効なmaskの個数分 (N*T)個以下の数で平均をとる
```
loss /= mask.sum()
```
